# Beach Hydrovac Website

Professional website for Beach Hydrovac LLC - A Division of Advanced Infrastructure Mapping.

## Quick Deploy to Vercel

### Option 1: Vercel CLI (Fastest)
```bash
# Install Vercel CLI if you haven't
npm i -g vercel

# Navigate to project
cd beachhydrovac-website

# Deploy
vercel
```

### Option 2: GitHub + Vercel Dashboard
1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com)
3. Click "Import Project"
4. Select your repository
5. Click "Deploy"

### Option 3: Drag & Drop
1. Go to [vercel.com](https://vercel.com/new)
2. Drag the entire `beachhydrovac-website` folder
3. Click "Deploy"

## Local Preview

```bash
# Open the HTML file directly in your browser, or:
python -m http.server 8000
# Then visit: http://localhost:8000
```
